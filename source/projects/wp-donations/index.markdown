###WP-Donations


####Project Type

Company Project (Zencap)

####Project Dates

November 2013 to January 2014


####Project Description

A Wordpress plugin that handles site subscriptions. It integrates with the Bitpay and Paypal API to handle processing of payments.

####Technologies Used

- HTML
- CSS 
- JavaScript
- PHP 
- Wordpress
- Paypal API
- Bitpay API
